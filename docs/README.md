| Folder | Purpose |
| --- | --- |
| `frontend/` | Dashboard UI, rendering, modals, and client-side state. |
| `backend/` | Python scraper, planned Flask backend, and automation. |
| `database/` | Supabase schema and file-based data contracts. |
| `integrations/` | Procurement portal, Telegram, GitHub Actions, and third-party services. |
